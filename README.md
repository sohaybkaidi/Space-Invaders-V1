# Space Invaders

A Space Invaders remake built with HTML5, CSS3, and JavaScript.

## Features

* Single-player and two-player modes
* Progressive waves
* Boss battles every 10 waves
* Multi-phase boss fights

## Controls

* Enter: Start
* P: Pause / Resume
* T: Toggle Two-Player Mode

### Player 1

* ← → Move
* Space: Shoot

### Player 2

* A / D: Move
* W: Shoot

## Purpose

This project was created to improve my HTML, CSS, and JavaScript skills. It is my first game project of this size, and I am proud of the result.
